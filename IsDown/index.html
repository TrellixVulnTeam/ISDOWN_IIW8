About Us

O nosso objetivo é notificar a comunidade académica sobre períodos de manutenção de sistemas e/ou equipamentos, bem como de indisponibilidades de serviço momentâneas.

Aplicação Web

Vamos interagir da mesma forma com os vários clientes do nosso serviço, ou seja através de uma aplicação web. A aplicação vai ser desenvolvida com React.

REST API

Este módulo é o núcleo do nosso sistema de informação, responsável pela comunicação com todos os outros componentes.Parte da informação vai ser obtida por via de API sobre o SCOM.

Continuous Integration / Continuous Deployment

Vamos usar a prática de CI/CD para desenvolver e implementar todos os nossos componentes.Para isso vamos usar Docker containers e pipelines do GitLab.
